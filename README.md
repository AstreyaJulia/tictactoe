# Учебный проект "Крестики-нолики"

<img src="https://github.com/AstreyaJulia/tictactoe/blob/master/preview.png" alt="Крестики-нолики" style="width:100%;"/>

Классическая логическая игра на поле размером 3х3 для 2-х игроков (с слабеньким ИИ).

[Страница проекта на GitHub Pages](https://astreyajulia.github.io/tictactoe/)

## Использовано
- HTML
- CSS
- JavaScript
- Сетка на Grid и Flex
